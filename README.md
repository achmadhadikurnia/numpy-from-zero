# NumPy From Zero

This repository documents my journey of learning NumPy from scratch.

The goal is not just to understand the theory, but to practice it through simple examples and gradually move into more practical use cases.

I created this repository because I believe learning becomes more effective when it is documented, and hopefully it can also help others who are learning NumPy.

Topics covered include:
- basic arrays
- indexing & slicing
- mathematical operations
- broadcasting
- and simple mini projects

This is an ongoing project and will continue to be updated.

If you're also learning NumPy, feel free to explore, fork, or discuss 🙌
